# A'dan Z'ye C# Programlama Dilini Öğreniyorum

Bu depo, **Murat Yücedağ**'ın **"Uygulama Geliştirerek C# Öğrenin: A'dan Z'ye Eğitim Seti"** eğitimini takip ederken gerçekleştirdiğim günlük çalışmaları, aldığım notları ve geliştirdiğim uygulamaları barındırmaktadır. 

Her gün öğrendiğim yeni kavramları, kod satırlarına dökerek bu günlük altında canlı tutuyorum.

---

## Takip Edilen Kaynak

* **Eğitmen:** Murat Yücedağ
* **Kurs Adı:** Uygulama Geliştirerek C# Öğrenin: A'dan Z'ye Eğitim Seti
* **Eğitim İçeriği:** Temel Programlama, Nesne Yönelimli Programlama (OOP), SQL & T-SQL Veri Tabanı Entegrasyonu, Form Tabanlı Geliştirme ve Proje Süreçleri.

---

## 📌 Yol Haritası ve Kurs İlerleme Durumu

Kurs müfredatına paralel olarak belirlediğim gelişim aşamaları:

- [ ] **Aşama 1:** C# Temelleri (Değişkenler, Operatörler, Karar Yapıları, Döngüler, Diziler)
- [ ] **Aşama 2:** Metotlar ve Nesne Yönelimli Programlama (OOP - Encapsulation, Inheritance, Polymorphism)
- [ ] **Aşama 3:** SQL & T-SQL ile Veritabanı ve C# İlişkilendirmesi (ADO.NET / Entity Framework)
- [ ] **Aşama 4:** Windows Form ile Dinamik Uygulamalar Geliştirme
- [ ] **Aşama 5:** Kurs Sonu Projesi / Bitirme Çalışması

---

## 🛠️ Kullanılan Teknolojiler & Araçlar

- **Dil:** C# (.NET Core / .NET Framework)
- **IDE / Veritabanı:** Visual Studio 2026, Microsoft SQL Server (İlerleyen derslerde)
- **Versiyon Kontrol:** Git & GitHub

## 💡 Projeyi Klonlama ve Çalıştırma

Bu depodaki çalışmaları kendi bilgisayarınızda incelemek veya çalıştırmak isterseniz aşağıdaki adımları takip edebilirsiniz:

### 1. Depoyu Bilgisayarınıza İndirin
Bilgisayarınızda projeyi kurmak istediğiniz boş bir klasörde terminal (Komut İstemi / Git Bash) açın ve şu komutu çalıştırın:
```bash
git clone [https://github.com/mehmetegok/CSharp-Bootcamp.git](https://github.com/mehmetegok/CSharp-Bootcamp.git)
```
2. Projeyi Çalıştırma
Visual Studio ile: Klonlanan klasörün içerisindeki .sln (Solution) dosyasını veya ilgili günün klasöründeki .csproj dosyasını çift tıklayarak Visual Studio 2026 ile doğrudan açabilir ve F5 tuşuna basarak çalıştırabilirsiniz.
